# 价值投资学习路径

交互式、离线可用的价值投资学习工具。6 个模块覆盖从投资心法到组合管理的完整路径。

**在线体验** https://jerryhub366.github.io/vi_system/

## 功能

- 6 个递进式学习模块：框架心法 → 生意理解 → 财报分析 → 单位经济 → 估值 → 风控
- 每个模块包含：学什么、怎么练、输出物、KPI 指标
- 关键词搜索过滤
- 完成进度追踪（localStorage 本地保存）
- 一键复制输出物清单
- 深色主题，响应式布局，移动端友好

## 使用

直接用浏览器打开 `index.html`，无需安装任何依赖。

## 技术

纯 HTML + CSS + JavaScript 单文件实现，零依赖。
